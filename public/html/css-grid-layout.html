Learn CSS Grid Layout


CSS Grid is one of the most powerful ways of creating Responsive websites, so if you want your website to look good both on a big Monitor screen, Laptop screen and all the way to a small Mobile phone’s screen, here are the necessary steps to implement CSS GRID



STEP 1

Create a HTML file and add a div container that will have the display property which will be set to Grid in the CSS file.


HTML file


< div class="container"></ div >

CSS file


.container{
    Width: 100%;
    Height: 100vh;
    background: #aaa;
    Display: Grid;
}

STEP 2

In the HTML file create three sections inside the container div with the class of section1, section2 and section3 respectively. Then in the CSS file, set different background for each section.


HTML file


< div class="section1"></ div >
< div class="section2"></ div >
< div class="section3"></ div >

CSS file


.section1{
    background: red;
}
.section2 {
    Width: green;
}
.section3{
    Width: blue;
}

STEP 3

In the CSS file, add the grid-template-columns to the container and set it to auto for each column. The three autos refers to three columns, two autos refers to two columns and so on. After adding the grid-template-column property, also add a grid-template-rows property to the container, and this property will set the specified height to the sections we have created.


.container{
    Width: 100%;
    Height: 100vh;
    background: #aaa;
    Display: Grid;
    grid-template-columns: auto auto auto;
    grid-template-rows: 50% 50% 50%;
}

What we did above is we added three columns using three autos and we also specified the height for the sections to 50% of the container.


STEP 4

Now that each section’s height has been specified and we also created the columns, we can go further with the properties that are associated with CSS GRID. Now in the CSS file, add property called Grid-gap to the container and set it to any value that you want.


.container{
    Width: 100%;
    Height: 100vh;
    background: #aaa;
    Display: Grid;
    grid-template-columns: auto auto auto;
    grid-template-rows: 50% 50% 50%;
    grid-gap: 50px;
}

What we did above is we have set the gap between each section to 50px away from each other.


STEP 5

Now there is an easy way to create the columns, instead of adding autos for each column, we can set or add a function called repeat() to the grid-template-columns property. Inside this functiuon we can specify the number of columns we want and that will be done for us.


.container{
    Width: 100%;
    Height: 100vh;
    background: #aaa;
    Display: Grid;
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: 50% 50% 50%;
    grid-gap: 50px;
}

We added three inside the function to tell the browser that we only want three columns inside the container. The 1fr specifies how we want to divede these columns, in our case we divided each column by 1. The fr stands for Fraction. Also we can replace the grid-template-rows to grid-auto-rows which does the exact same thing but in will make it easy to set the value for each section. So instead of saying 50% three times for each column, we are going to say 50% only once as the value for grid-auto-rows

The 1fr specifies how we want to divede these columns, in our case we divided each column by 1. The fr stands for Fraction. .container{
    Width: 100%;
    Height: 100vh;
    background: #aaa;
    Display: Grid;
    grid-template-columns: repeat(3, 1fr);
    grid-auto-rows: 50%;
    grid-gap: 50px;
}

STEP 6

Now that we have some of the properties that are associated with CSS GRID, we can now go further in learning some of them.
In the CSS file, add two properties called grid-columns and grid-rows to each section. This two properties specifies where you want to place your sections inside the container. By default, each section is place the way you added them respectively inside the HTML file.

The grid-colums property has lines that go vertically while the grid-rows has line that go horizontally. When we set the grid-column to 1, we are saying we want the section to start at column 1 and stop at column 2, if we set it to 2 it will sto at 3 and so on. When we set the grid-rows to 1, we are saying we want a section to start at row 1 and end at row 2, and if we set it to 2 it will stop at row 3 and so on.


.section1{
    background: red;
    Grid-columns: 1;
    Grid-rows: 1;
}
.section2 {
    Width: green;
    Grid-columns: 1;
    Grid-rows: 1;
}
.section3{
    Width: blue;
    Grid-columns: 1;
    Grid-rows: 1;
}

STEP 7

Now I that we have specified the values for the grid-columns and grid-rows, we can now go further and add a stop or and end point for the sections. To specify a stop or end point, we add a forward slash followed by the value of the column or row we want to end or stop the section at.


.section1{
    background: red;
    Grid-columns: 1 / 2;
    Grid-rows: 1 / 2;
}
.section2 {
    Width: green;
    Grid-columns: 2 / 3;
    Grid-rows: 1 / 2;
}
.section3{
    Width: blue;
    Grid-columns: 3 / 4;
    Grid-rows: 1 / 2;
}


That’s it, you will now be able to create responsive websites using CSS GRID. Thank you for reading and visiting my Website!.
